# Kantin Negro

Ujian Kompetensi Keahlian tahun ajaran 2024/2025(the code lil bit messy)

## Inspired from

- [Amici](https://amici-restaurant.com)
- [Brews & Grooves](https://www.brewsandgrooves.com)

## How to start

### Install Dependency

```bash
pnpm install
```

### Setup .env file

### Run the server

```bash
pnpm dev
```
